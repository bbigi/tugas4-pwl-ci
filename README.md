# 📝 Website Manajemen Blog Sederhana (Tugas PWL Pertemuan 4)

Proyek ini adalah tugas kelompok untuk mata kuliah **Pemrograman Web Lanjut** yang mengimplementasikan konsep **MVC (Model-View-Controller)** pada framework **CodeIgniter 4**, dengan fokus utama pada pemisahan logika di Controller dan tampilan di View.

## 🛠️ Struktur Teknis
* **Framework** : CodeIgniter 4.
* **Controller**: `application/controllers/Blog.php` sebagai pusat logika dan pengatur data.  
* **Data**: Menggunakan data statis (array) sesuai instruksi soal untuk mensimulasikan database.
* **View**: Menggunakan PHP Native untuk menampilkan data dinamis dari Controller ke HTML.
